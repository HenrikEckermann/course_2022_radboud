Microbiome data science with R/Bioconductor

2 Program

The course takes place daily from 9am – 5pm (CEST), including coffee and lunch breaks. See Detailed program.

We expect that participants will prepare for the course in advance, see section 4. Online support is available.

The material follows open online book created by the course teachers, Orchestrating Microbiome Analysis https://microbiome.github.io/OMA. This is R/Bioconductor framework for microbiome data science.
